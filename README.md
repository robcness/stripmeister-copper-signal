# StripMeister Copper Recovery Signal

Private first-iteration prototype for an embeddable StripMeister website widget.

## Purpose

The widget gives copper wire recyclers, scrappers, electricians, and contractors a simple at-a-glance indicator that connects copper-market conditions to a practical StripMeister next step.

## Current CTA destinations

- Calculate your wire recovery: https://www.stripmeister.com/pages/scrap-calculator
- Compare StripMeister models: https://www.stripmeister.com/pages/compare-machines
- Find your StripMeister model: https://www.stripmeister.com

## Prototype scoring model

- Price momentum: 35%
- Five-year value context: 25%
- Recycling demand relevance: 20%
- Short-term risk adjustment: 20%

This is intentionally framed as a marketing decision aid, not a copper forecast, financial advice, scrap-yard pricing guarantee, or ROI guarantee.

## Suggested embed direction

For a Shopify/GitHub Pages style embed, this can be bundled as a single iframe or converted into a single self-contained HTML snippet after the messaging and scoring logic are approved.
